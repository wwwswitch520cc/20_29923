<p>前秩序 Exorder 是Solid9 Studio制作的一款奇幻题材的战棋游戏，游戏将提供单人战役和多人遭遇战模式。</p>
<p>单人战役将描绘一个“前秩序”Exorder的世界，带来大量富有挑战和剧情意义的任务。</p>
<p>遭遇战模式将允许两名玩家在预先设计好的地图上进行对决。</p>
